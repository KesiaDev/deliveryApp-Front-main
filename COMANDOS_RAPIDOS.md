# ⚡ Comandos Rápidos - Testar o App

## 🚀 Sequência de Comandos (Copie e Cole)

```bash
# 1. Ir para a pasta do projeto
cd C:\Users\User\Desktop\deliveryApp-Front-main\deliveryApp-Front-main

# 2. Instalar dependências
flutter pub get

# 3. Verificar erros
flutter analyze

# 4. Ver dispositivos disponíveis
flutter devices

# 5. Rodar o app
flutter run
```

---

## 📱 Comandos por Plataforma

### Android
```bash
flutter run -d android
```

### Web (Chrome)
```bash
flutter run -d chrome
```

### Windows Desktop
```bash
flutter run -d windows
```

---

## 🔧 Comandos de Manutenção

### Limpar projeto
```bash
flutter clean
flutter pub get
```

### Verificar versão do Flutter
```bash
flutter --version
```

### Ver dispositivos conectados
```bash
flutter devices
```

---

## ⌨️ Comandos Durante Execução

Quando o app estiver rodando, pressione:

- **`r`** = Hot Reload (recarregar)
- **`R`** = Hot Restart (reiniciar)
- **`q`** = Quit (fechar)
- **`h`** = Help (ajuda)

---

